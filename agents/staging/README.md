# dACP-agents-registry - Staging Environment

The Boson dACP *staging* environment allows developers to test and experiment with BOSON dACP features without using real cryptocurrencies, providing a safe space to identify bugs, try new features and perform demos. It helps ensure that applications function correctly before being deployed on the production envionment and mainnet chains.
